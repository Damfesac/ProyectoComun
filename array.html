Array

Definición:

Un array es una estructura que utilizamos para almacenar objetos de un mismo tipo en un grupo.

Pueden ser dinámicos o estáticos

Dinamica: Son arrays que no tienen un tamaño definido y podemos declarar su tamaño al incializar el objeto.

Se definen de la siguiente manera:

Object[] nombreobjeto = new Object[10]

Estatica: Se crean con el valor ya inicializado directamente

Se definen de la siguiente manera:

d

Object nombreobjeto[10]

ArrayLists

La claseArrayListpermite almacenar datos en memoria de forma similar a los arrays, con la ventaja de que el número de elementos que almacena lo hace de forma dinámica, es decir: no es necesario declarar su tamaño como pasa con los arrays brutos.
Otra cosa muy importante a la hora de trabajar con los arrayList son los "iteradores". Los iteradores sirven para recorrer los arraylist y poder trabajar con ellos. Los iteradores solo tienen tres métodos: que son el "hasNext()" para comprobar que siguen quedando elementos en el iterador, el "next()" para que nos de el siguiente elemento del iterador; y el "remove()" que sirve para eliminar el elemento del Iterador.
