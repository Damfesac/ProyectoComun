Array y ArrayLists

- Array

• Un array es una estructura que utilizamos para almacenar objetos de un mismo tipo en un grupo usando posiciones de memoria contiguas.

• Esto permite que se pueda acceder a la informacion de forma rapida y eficaz.

•Su declaración e inicialización es la siguiente:

- Object nombre_array[]

- nombre_array= new Object[tamaño]

•Para asignarle valores, tan solo tenemos que recorrer el array e ir asignando a cada posicion un valor

- for(int i=0;i < tamaño;i++){

nombre_array[i]=valor

}

ArrayList:

•Un ArrayList es una clase de java que permite definir arrays dinámicos, es decir, que pueden modificar su tamaño en base a los elementos que se le vayan introduciendo o quitando

•Su declaracion es la siguiente:

- ArrayList <Object> lista=new ArrayList <Object>()

•Para insertar valores:

- lista.add(valor)

A medida que vamos insertando valores, el tamaño del array aumenta

•Para acceder a valores dada una posicion i:

- lista.get(i)

•Para cambiar el valor de la posicion i:

- lista.set(i,valor2)

•Para quitar el elemento de la posicion i:

- lista.remove(i)

A medida que vamos removiendo valores, el tamaño del array disminuye

•Para quitar todos los elementos del arraylist:

- lista.clear()

•Obtener el tamaño actual del arraylist:

- lista.size()
